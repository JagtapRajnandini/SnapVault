# SnapVault

SnapVault is a Flask-based web application that helps users organize screenshots and images using OCR and AI.

## Current Status

🚧 Project is under active development.

## Planned Features

- User authentication
- Image upload
- OCR text extraction (EasyOCR)
- Automatic image categorization
- Search screenshots by extracted text
- Smart reminders
- AI-powered content understanding
- Personal dashboard

## Tech Stack

- Python
- Flask
- SQLAlchemy
- SQLite
- EasyOCR
- HTML
- CSS
- Bootstrap
